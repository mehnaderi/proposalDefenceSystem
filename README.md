# **Proposal Defense System**

## **How to run the project:**

1. Install Vuejs by npm: (for more information visit Vuejs website: )

```bash
# latest stable
$ npm install vue
```

[https://vuejs.org/v2/guide/installation.html](https://vuejs.org/v2/guide/installation.html)

1. Run the project using: (wait for the the dependencies to be installed)

```bash
npm run serve
```
